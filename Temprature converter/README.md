# 🌡️ __Python Temperature Converter (CLI)__

A simple command-line temperature converter built using Python.  
This project converts between different temperature units using clean logic and basic functions.

---

## 🚀 Features

- *Convert between:*
  - Celsius (C)
  - Fahrenheit (F)
  - Kelvin (K)
  - Rankine (R)
- Uses a base unit (Celsius) for accurate conversions
- Colored terminal output using colorama
- Basic error handling for invalid inputs

---

## 📸 Example

- __Simple Temperature Converter__

 Enter the Temperature: 100

Enter the unit: C

Enter conversion unit: F

*Converted temperature: 212.0 F*

## ⚠️ Limitations

- Runs only once (no loop yet)
- Limited to basic temperature units
- No input validation beyond basic checks
